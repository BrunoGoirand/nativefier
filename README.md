This repository is a fork of
[nativefier/nativefier](https://github.com/nativefier/nativefier).

The upstream project stopped being actively developed and has now been archived.
That is unfortunate, because Nativefier remains a very useful solution for
turning web sites into desktop applications with minimal friction.

This fork exists because I want to keep Nativefier usable, maintained, and able
to evolve. The goal is to preserve what made the original project practical
while updating it where needed for current Electron, Node.js, operating system,
and web platform changes.

The original upstream README has been preserved in [UPSTREAM.md](UPSTREAM.md).
For available command-line options, see [API.md](API.md).
